Выберите все подходящие ответы из списка:

1. Даже для программы состоящей из одной пустой функции int main() { return 0; } все равно требуется линковка.

2. Если в коде C++ вы вызываете функцию, которая была объявлена, но не была определена, то это ошибка этапа компиляции.

3. Для программы "Hello, World!", состоящей всего из одного файла, не требуется линковка.

4. Если в коде C++ вы вызываете необъявленную функцию, то это ошибка этапа компиляции.

5. Если в коде C++ вы вызываете необъявленную функцию, то это не ошибка, при условии, что функция была где-то определена.

6. Если в коде C++ вы вызываете функцию, которая была объявлена, но не была определена, то это ошибка этапа линковки.

Верные ответы: 1, 4, 6.
